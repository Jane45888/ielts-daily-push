# 雅思口语每日推送 📬

> 每天自动发一道雅思口语练习题到你的邮箱，帮你保持练习节奏，冲刺 Band 7+！

**零成本 · 零门槛 · 不用装任何软件 · 手机就能配**

---

## 它是怎么工作的？

```
每天北京时间 17:40
       ↓
GitHub 云端自动运行
       ↓
从题库随机抽题（Part 1 + Part 2/3）
       ↓
发送精美邮件到你的 QQ 邮箱
       ↓
打开邮箱 → 开始练习 🎯
```

你不需要开着电脑，不需要写代码，配好之后就全自动了。

## 🚀 5 分钟配置教程

### 第一步：注册 GitHub 账号（如果已有请跳过）

1. 打开 [github.com](https://github.com)，点击右上角 **Sign up**
2. 填写邮箱、设置密码、用户名
3. 验证邮箱 → 完成！

> 💡 如果你已经有 GitHub 账号，直接登录就行。

---

### 第二步：获取 QQ 邮箱授权码

这一步很关键，授权码就是让程序帮你发邮件的"钥匙"。

1. 打开 [mail.qq.com](https://mail.qq.com) 并登录你的 QQ 邮箱
2. 点击顶部 **设置**（齿轮图标）→ 左侧选择 **账户**
3. 往下滑，找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务** 这一块
4. 点击 **IMAP/SMTP 服务** 后面的 **开启**
5. 按提示用手机发短信验证
6. 验证成功后，页面会显示一个 **16 位授权码**（类似 `abcdefghijklmnop`）
7. **立刻复制保存下来**，后面要用！

> ⚠️ 授权码只显示一次，请务必复制保存好！

---

### 第三步：在 GitHub 创建你的仓库

1. 登录 GitHub 后，打开 [github.com/new](https://github.com/new)
2. 填写：
   - **Repository name**（仓库名）：`ielts-daily-push`
   - **Description**（描述）：雅思口语每日推送（随便写）
   - 选择 **Public**（公开）
3. **不要勾选**任何初始化选项（不要勾 Add a README、.gitignore 等）
4. 点击绿色按钮 **Create repository**

---

### 第四步：上传项目文件

仓库创建好后，你会看到一个空仓库页面。现在把文件传上去：

**方法一：直接上传（推荐新手）**

1. 在空仓库页面，点击 **Add file** → **Upload files**
2. 把本项目的以下 **4 个文件** 拖进去：
   - `send_email.js`
   - `ielts_topics.json`
   - `package.json`
   - `.github/workflows/daily-ielts.yml`
   > ⚠️ `.github` 是一个文件夹，上传时需要保持文件夹结构。如果网页上传不好操作文件夹，可以用方法二。

3. 在页面底部的输入框填写 commit message：`Initial commit`
4. 点击绿色按钮 **Commit changes**

**方法二：通过 ZIP 上传**

1. 在空仓库页面，点击 **Add file** → **Upload files**
2. 把整个项目文件夹打包成 ZIP 文件，拖进去
3. 填写 commit message → **Commit changes**
4. 然后进入仓库，把 ZIP 文件删除（只保留其他文件）

**方法三：用手机操作**

如果你在手机上，也可以：
1. 先在电脑上把文件下载/保存好
2. 用手机浏览器打开 GitHub，登录后同样可以上传文件

---

### 第五步：配置邮箱密码（Secrets）

这是最后一步，把你的邮箱和授权码告诉 GitHub（它们会被加密保存，别人看不到）。

1. 进入你的仓库页面 `github.com/你的用户名/ielts-daily-push`
2. 点击顶部的 **Settings**（设置）
3. 左侧菜单找到 **Secrets and variables** → 点击展开 → 点击 **Actions**
4. 点击右侧绿色按钮 **New repository secret**

**添加第 1 个 Secret：**

| 字段 | 填写内容 |
|------|----------|
| **Name** | `SMTP_USER` |
| **Secret** | 你的 QQ 邮箱地址，例如 `123456789@qq.com` |

填好后点击 **Add secret**

**添加第 2 个 Secret：**

再次点击 **New repository secret**

| 字段 | 填写内容 |
|------|----------|
| **Name** | `SMTP_PASS` |
| **Secret** | 第二步获取的 16 位授权码 |

填好后点击 **Add secret**

**添加第 3 个 Secret（可选）：**

> 如果你想发到别的邮箱，就加这个。不填的话默认发到你自己的 QQ 邮箱。

| 字段 | 填写内容 |
|------|----------|
| **Name** | `EMAIL_TO` |
| **Secret** | 你想接收邮件的邮箱地址 |

---

### 第六步：测试一下！

1. 回到仓库主页
2. 点击顶部的 **Actions** 标签
3. 左侧会显示 **雅思口语每日推送**，点击它
4. 点击右侧 **Run workflow** → 再点 **Run workflow** 确认
5. 等待 30 秒左右，刷新页面，看到绿色 ✅ 就成功了！

然后去你的 QQ 邮箱收件箱（或垃圾邮件）查看，应该能收到一封精美的雅思练习邮件 📬

## ⚙️ 想调整推送时间？

编辑 `.github/workflows/daily-ielts.yml` 文件，修改 cron 表达式：

```yaml
schedule:
  - cron: '40 9 * * *'   # 北京时间 17:40
```

常用时间对照：

| 北京时间 | cron 表达式 |
|----------|------------|
| 07:00 | `0 23 * * *` |
| 08:00 | `0 0 * * *` |
| 09:00 | `0 1 * * *` |
| 12:00 | `0 4 * * *` |
| 17:40 | `40 9 * * *` |
| 20:00 | `0 12 * * *` |
| 22:00 | `0 14 * * *` |

> 💡 GitHub 的 cron 使用 UTC 时间，北京时间 = UTC + 8 小时。

## ❓ 常见问题

**Q: 邮件收不到怎么办？**
- 检查垃圾邮件/广告邮件文件夹
- 确认 SMTP_USER 和 SMTP_PASS 填写正确
- 确认授权码没有过期或被重置

**Q: 可以用 163/其他邮箱吗？**
- 目前只支持 QQ 邮箱（SMTP 服务器写死了 smtp.qq.com）
- 如果需要支持其他邮箱，需要修改 send_email.js 中的 SMTP 配置

**Q: 每天的题目会重复吗？**
- 是随机抽取的，有可能会重复，但题库有 70+ 道题，重复概率不高

**Q: 可以自定义题库吗？**
- 可以！编辑 `ielts_topics.json` 文件，按相同格式添加/修改题目即可

**Q: 怎么停止推送？**
- 方法 1：删除仓库
- 方法 2：在 Actions 页面点击 "Disable workflow"
- 方法 3：删除 Secrets，程序会自动报错停止

**Q: GitHub Actions 免费吗？**
- 免费账户每月有 2000 分钟的免费额度，这个项目每次运行不到 1 分钟，完全够用
